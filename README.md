# Real_Time_Detection

This is Real Time Face Detection using CNN

## step 1 : 
```
git clone https://github.com/amantiwari1/Real_Time_Detection.git
```

## step 2 : 
```
cd Real_Time_Detection
```

## step 3 : 
```
pip install -r requirements.txt
```

## step 4 : 
```
python setup.py
```
